# AI Systems Roadmap

我从神经网络基础到大模型推理系统的学习记录。
每学完一项就把 `[ ]` 改成 `[x]`，GitHub 会自动显示进度。

## 仓库结构

| 目录 | 放什么 |
|------|--------|
| `notes/` | 每个主题的学习笔记（一篇一个 `.md`） |
| `code/` | 自己动手实现的代码（micrograd、attention …） |
| `papers/` | 读论文的**笔记**（不是论文 PDF 本身） |
| `projects/` | 阶段性的小项目 / 综合实践 |
| `progress.md` | 每周进度、投入时间、反思 |
